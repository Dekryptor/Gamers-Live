Features Take a look at what we offer

    Gamers Live is easy to install with the WEB interfaced installer that follows, you will be up and running in less then 5 minutes!

  • Analyse

    The integrated admin panel allows admins to monitor their sites. See when there are most viewers & streamers, we also have easy integration with Google Analytics!

    You can completely manage your site, and suspending or changing users are a flick of a switch!

    Security is the key, and all your streams will be highly secured with client specific stream keys, so only the rightfully owner can stream on his channel!.

Pages

  • Frontpage

    The front page shows all the online streaming featured streamers, and list the 5 highest ranked ones (as an admin you will give each featured streamer an level, and this level determines if they are showed first or last.)

  • Search

    With the integrated search bar at the top right corner of every page users will have no problem by finding exactly what they want. Whether its a specific game they wish to view or a specific streamer, the smart search helps them towards finding what they need.

  • Channel Pages

    The channel is designed to be easy and clean. But still have all the features as a dedicated chat module, statistics, streamer information, ADS, tipping feature and costumizability. User can at any time change title, game details etc. Also custom images for each user is avalible, and users can upload offline img, avatar etc.

  • Chat Module

    We have built a custom made chat system that automatically enables chat on every channel! Both side my side with the live steam but also windowed mode are supported.

  • Chat Moderation

    We have also build a tool to manage the chat, here clients will be able to ban or add moderators to the chat of their channel.

  • Admin CP

    As an admin you can easily manage your website and see live statistics of what is going on. Banning and changing settings for each client is as simple as clicking a link on their channel page.
